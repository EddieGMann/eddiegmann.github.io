Qualtrics

  • Course Evaluations
    • Create, send and summarize responses for all CRTP classes and BIOSTAT MB Courses
    • Course evaluation summaries are available in the shared drive. Course evaluation summaries need to look rather similar year to year, so that program directors can easily identify changes. I tend to copy and paste the evaluation from last year and update the information. Make sure to title the new evaluation with the year somewhere in there.
    • Course Evaluation Generator
      • This is a python program that will take the Qualtrics csv and automatically generate an evaluation summary that looks similar enough to previous years. Drag and drop the "data & analysis" csv into the generator. Remember to read through the new evaluation and make sure that everything looks normal.
  • Surveys to gather personal information
    • T-shirt size, if they want a ticket to the state fair, how many tickets for graduation, etc.
  • Course director's specials
    • Time slot sign-ups for in-person speaking final
    • Group project peer evaluations where they only see the names of group members
